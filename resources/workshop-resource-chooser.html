Suggested path

A simple attendee sequence

Prepare a safe sandbox. Use the pre-workshop checklist if you are not sure what project, sample input, or verification signal to bring.

Route credential confusion. Use the credential routing Q&A if login, account, API-key, environment, or policy issues appear before the first run.

Open the harness map. Anchor on why the surrounding workflow matters as much as the model.

Add project rules. Use the CLAUDE.md cheat sheet to make the next agent run more deliberate.

Package repetition. Build a first skill, then run Skill Wizard as a quick quality gate.

Shape an idea. Generate a product brief, ask only high-impact missing questions, then draft a PRD/OpenSpec starter.

Choose one follow-up workflow. Use the post-workshop action plan to commit to a small safe project with verification evidence.

Review before scaling. Improve prompts and review PRDs before moving to optional advanced demos.
