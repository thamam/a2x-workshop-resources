Prompt improver.

Turn a rough request into clearer instructions. The key move: rewrite negative corrections into positive behavior the model can follow.

Copyable prompt

You are my prompt improver. Improve the prompt below so it is clearer, more actionable, and less ambiguous.

Rules:
- Preserve the user's intent.
- Convert negative corrections into positive desired behavior when possible.
- Add missing context as placeholders, not invented facts.
- Separate goal, context, constraints, output format, and quality bar.
- Keep the improved prompt concise enough to use.

Rough prompt:
[PASTE PROMPT]

Return:
1. Improved prompt
2. What changed and why
3. Any missing context worth asking for

Example

Instead of “don’t make it corporate,” ask for “direct, practical language with a warm expert tone; prefer concrete examples over marketing phrasing.”

Use when

An agent produces something almost right but drifts in tone, scope, structure, or assumptions.

Upgrade path

Save the improved version as a reusable workflow, project rule, or skill when it works repeatedly.
