Safe scope

Pick a starter workflow that can survive a messy first attempt.

Good first workflows are useful, reversible, and easy to inspect. Avoid production credentials, payment flows, private client material, and irreversible automation until the harness is proven.

24-hour plan

Three steps before momentum evaporates

1 · Name the loop

Choose one repeated annoyance.

Examples: summarize customer notes, draft a spec from a brief, turn support themes into issues, or review a small code change against team rules.

2 · Build the harness

Add rules, examples, and stop conditions.

Use a small CLAUDE.md, one safe sample input, expected output shape, and “ask before guessing” criteria before asking the agent to act.

3 · Verify the output

Demand evidence.

Finish with a rendered page, passing test, diff, checklist, or exported review packet. If there is no artifact, the workflow is not learned yet.

Starter workflow brief

  • Workflow: The repeated task I want to improve is…
  • Why it matters: It costs time or creates mistakes because…
  • Safe sample input: Use fictional or sanitized material only…
  • Expected output: The artifact should look like…
  • Stop conditions: Ask me before touching production, secrets, money, legal/health data, or external messages.
  • Verification: Prove the result with…

Prompt to run

“Help me turn this repeated workflow into a safe first Claude Code project. First, identify missing context and risky assumptions. Then propose a tiny reversible version, the files or notes it needs, and the exact verification we should run before calling it done. Do not implement until the scope is clear.”

Escalation path

When the first loop works

Make it reusable

If you repeat the same prompt twice, turn the successful steps into a skill or project rule.

Add review gates

For code or specs, add an OpenSpec-style proposal before implementation and a review packet after changes.

Bring it back to the team

Share the small artifact, not a theory. A working example is easier to critique, improve, and adopt.
