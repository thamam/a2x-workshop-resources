Workshop flow

How to use this hub live

1. Open with the harness idea.

Show that better agent results come from environment, rules, tools, feedback loops, and review gates.

2. Give attendees rules they can use today.

Send them to the CLAUDE.md cheat sheet before live coding.

3. Turn repetition into infrastructure.

Use the first-skill guide when moving from tips to reusable workflows.

4. Hand out practical prompt tools.

Use the product brief generator, interviewer, prompt improver, and PRD starter when attendees start shaping their own ideas.

5. Park advanced demos safely.

Use placeholders for tools until public review is complete.
